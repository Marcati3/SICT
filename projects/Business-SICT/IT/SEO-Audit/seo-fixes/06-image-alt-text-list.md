# Image Alt Text Recommendations
## www.sic.co.th Homepage

**WHERE:** WordPress Admin > Media Library (or edit via Flatsome/Elementor page editor)
**IMPACT:** Fixes the biggest SEO issue — 87% of images currently have no alt text

### Rules
- Keep alt text under 125 characters
- Describe the image truthfully
- Include one relevant keyword naturally — do not stuff
- Decorative images (separators, spacers) should keep alt="" (empty)
