## Key Rules
1. **Always read PROJECT-BRIEF.md** before starting any work on this project
2. **Product names must be exact** — Use SIC278, SIC279, SIC7150/51/52, SIC73F1, SIC61xx, SIC61AU, SIC43NT, SIC43S1, SIC4310, SIC4341, SIC4343
3. **Company context matters** — SIC is a Thai fabless IC design house (SET-listed), not a generic chip distributor. Outreach must reflect this positioning
4. **Tone for US outreach** — Professional, concise, value-led. No overselling. Lead with technical differentiation (read range, sensor interface uniqueness, full IP ownership)
5. **Priority order** — Follow the priority table in PROJECT-BRIEF.md (Merck #1 through Abbott #13)
6. **Track status** — Each company meeting should be tracked as: Confirmed / Outreach Sent / Researching / Not Started
